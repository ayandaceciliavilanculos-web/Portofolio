"# Portofolio" 
 Portfólio Pessoal

Este é o meu portfólio pessoal, desenvolvido para apresentar quem sou, as minhas competências, os meus projectos e formas de contacto. O objetivo é criar uma presença digital profissional e moderna.


##  Sobre o Projeto

O portfólio foi desenvolvido como parte do módulo de Programação Web.  
Ele apresenta:

- Informações pessoais e profissionais  
- Habilidades e competências  
- Projetos realizados  
- Formulário de contacto  
- Design moderno e responsivo  


## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Git & GitHub**


## Funcionalidades

- Layout responsivo  
- Secção "Sobre Mim"  
- Lista dos meus projectos  
- Formulário funcional com validação  
- Links para redes sociais
